needs:
1 piezo disc (at least 27mm)
arduino pro-micro
1 megaohm resistor

wiring:
piezo disc black cable to arduino gnd pin
piezo disc red cable to arduino A0 pin
1 megaohm resistor from gnd to A0 pin

run the code, keep the serial monitor screen on at all times
